# Candy Vending Machine Client

Этот проект реализует клиент для взаимодействия с сервером автоматического вендинг-машины конфет. Клиент отправляет запросы на покупку конфет и выводит ответ от сервера в стандартный вывод.

## Описание

Клиент принимает аргументы командной строки для создания запроса к серверу. Он сериализует данные в формат JSON, отправляет запрос через защищенное соединение TLS и выводит ответ от сервера.

## Структура проекта

- `client.go`: Основной файл с реализацией клиента.

## Функциональность

1. **Сбор данных из командной строки**: Клиент принимает параметры для типа конфет, количества конфет и количества денег.
2. **Сериализация данных**: Данные преобразуются в формат JSON.
3. **Настройка TLS**: Клиент использует сертификаты для установления защищенного соединения.
4. **Отправка запроса**: Клиент отправляет POST-запрос на сервер и получает ответ.
5. **Вывод ответа**: Ответ сервера выводится в стандартный вывод.

## Зависимости

- Go 1.18+
- TLS сертификаты для защищенного соединения

## Флаги командной строки

- `-k` (обязательный): Двухбуквенное сокращение для типа конфет (например, "CE").
- `-c` (обязательный): Количество конфет для покупки.
- `-m` (обязательный): Количество денег, переданных в запросе.

Пример использования:

```bash
go run client.go -k CE -c 3 -m 50
```

Этот пример отправляет запрос на покупку 3 конфет типа "CE" с 50 деньгами.

## Сертификаты

Клиент использует сертификаты для установления защищенного соединения с сервером. Сертификаты хранятся в директории `../certs/candy.tld/`:
- `cert.pem` — Сертификат клиента.
- `key.pem` — Ключ клиента.
- `cert.pem` — Сертификат CA (корневого удостоверяющего центра), используемый для проверки сервера.

### Загрузка сертификатов

Перед запуском клиента убедитесь, что сертификаты и ключи находятся в указанной директории и правильно настроены.

## Примечания

- Убедитесь, что сервер доступен и правильно настроен для приема запросов.
- Клиент требует действительный сертификат и ключ для подключения.
```
